## 🎯 Overview

**Task Manager Pro** is a sleek, full-stack web application that revolutionizes how you manage your daily tasks. Built with cutting-edge technologies and featuring a stunning dark-themed UI with glassmorphism effects, animated gradients, and smooth transitions.

### 🌟 Why Choose Task Manager Pro?

- 🎨 **Stunning Visual Design** - Modern glassmorphism UI with animated gradients
- ⚡ **Lightning Fast** - Optimized performance with minimal load times  
- 📱 **Fully Responsive** - Perfect experience on desktop, tablet, and mobile
- 🛡️ **Reliable** - File-based storage ensures your data is always safe
- 🎭 **Interactive** - Smooth animations and hover effects throughout

## 🚀 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Git

### 🔧 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/sumanisfr/Backend-Add_Task_-_Read.git
   cd Backend-Add_Task_-_Read
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   node server.js
   ```

4. **Open your browser**
   ```
   http://localhost:3000
   ```

### 🌐 Deployment on Render

1. **Fork this repository**
2. **Connect to Render**
3. **Configure build settings:**
   - **Build Command:** `npm install`
   - **Start Command:** `node server.js`
   - **Node Version:** 18.x
